# Prompts

Prompt templates are organized by capability under `backend/llm/prompts`.

- `analytics/`: analysis and trend prompts
- `insights/`: insight generation prompts
- `dax/`: DAX generation, explanation, and debugging prompts
- `documentation/`: report documentation prompts
- `executive/`: executive summary prompts

Agent-specific prompts live beside each agent in `backend/agents/*/prompts.py`.
