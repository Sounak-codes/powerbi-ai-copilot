# Evaluation

Evaluation coverage is organized under:

- `backend/evaluations/hallucination_tests`
- `backend/evaluations/regression_tests`
- `backend/evaluations/prompt_tests`
- `backend/evaluations/benchmark_suite`
- `backend/rag/evaluation`

Recommended checks:

- Groundedness against provided visual context
- Intent classification accuracy
- Retrieval recall and MRR
- DAX generation validity
- Regression tests for known report scenarios
